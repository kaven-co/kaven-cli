# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of `kaven-cli` seriously. If you suspect you have found a vulnerability, please report it immediately:

1. **Do not** open a public GitHub issue.
2. Email our security team at **security@kaven.com** (or equivalent contact).
3. Provide as much detail as possible to reproduce the issue.

We will acknowledge your report within 48 hours and provide an estimated timeframe for a fix. We will credit you in our release notes if the report leads to a fix.
